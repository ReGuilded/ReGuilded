# Description
The settings menu to handle enabling, disabling, and configuring addons and themes. Temporarily requires this addon, but will be in the main client eventually.

# Installation
### NOTE - Currently requires the addonLib addon. This will likely be built-into ReGuilded in the future.

Install the [addonLib](https://github.com/Metalloriff/ReGuildedAddons/tree/main/addonLib) addon.

Then simply place `main.js` in `%AppData%/.reguilded/settings/addons/SettingsInjector`.
